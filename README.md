# **Sistema de Gestión de Cuartos y Emisión de Boletas para Hospedaje Cocharcas**

---

## Descripción
Este proyecto tiene como objetivo mejorar la administración del Hospedaje Cocharcas mediante un sistema desarrollado en Java, que permite gestionar la asignación de cuartos y la emisión de boletas de pago. La aplicación facilita la organización de reservas, optimiza la facturación y reduce errores administrativos.

## Instalación
1. **Requisitos previos**:
    - Asegúrate de tener **Java JDK** (última versión recomendada: 23) y **NetBeans IDE** instalados en tu equipo.

2. **Clona el repositorio**:
    ```bash
    git clone https://github.com/Leandropacsi27/ProyectoHospedaje.git
    cd ProyectoHospedaje
    ```

3. **Importa el proyecto en NetBeans**:
    - Abre NetBeans.
    - Selecciona `File > Open Project` y navega hasta la carpeta del proyecto.
    - Abre el proyecto para cargar todos los archivos y configuraciones.

4. **Compila y ejecuta el proyecto**:
    - Haz clic en el botón `Run` de NetBeans o utiliza `Shift + F6` para compilar y ejecutar el proyecto.
